Let's do the following exercises with a partner. Grab a couple new neighbors and together go through each of these exercises one at a time. At the end we will ask different groups to present their solutions to each of the challenges and to explain why that is the right answer.

What's wrong with this code?

There are a few things wrong with this sample if-else statement. Can you find them all? Try to fix them all first and then try to run the code to check your work. 

if myVariable = 10 {
  console.log("myVariable was equal to 10")
else {
  console.log("myVariable was NOT equal to 10")
}

For each of the following exercises, make a variable at the top and use prompt() to ask for user input, and then write some if statements!

Log Greetings

Write a program that asks the user for their name and logs a different greeting for various different names. Handle the case of an unexpected name in a user-friendly way.

Primary Colors

Ask the user for a color and log whether the color is a primary color or not.

Even or Odd

Prompt the user for an integer and log if the integer is even or odd.
Hint 1: Google JavaScript's modulo operator.
Hint 2: What is the type of the user inputted value? If you need to convert this, Google how to do so.
