Activity: Logging to the Console

Go to repl.it. In the left box put your JavaScript code, press "run" and view the output in the right box.

Let's log (another word for "write") our name to the console: 

console.log("Ben")
console.log('Ben Sussman')
console.log('Sussman')
console.log("What's your name?")

Some important things to note:

  • My name is wrapped in quotes. That's to tell JavaScript that this is a String and not a variable name
  • You can use either " or ' in your JavaScript to create a String. Here is a great resource on JS Strings. Challenge Question: In what situations would you prefer to use a " or '?
  • You can learn in more detail about Chrome's console here: Using the Chrome Console

Saving your work in Repl.it

In order prevent losing work, we recommend logging in to repl.it to save your work. Another smart strategy is to copy code into well named files in your ~/code/ folder for future reference. Do what works for you, but make an effort to keep your code organized and you won't regret it! 

console.clear()
// code code code code
// all your code goes here
console.log("Made it to the end of our program")

Great! Now we're prepped and ready to learn some real programming!
