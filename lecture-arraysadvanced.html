Looping over Arrays

Let's return to the example that started off this topic: how could you calculate the average of a very large number of values, or of a dynamic number of values? Using variables like a1, a2, a3, etc won't work because we can't add these up in a loop. But this is where arrays come in!

So far, we have used hard-coded values for our array indexes. Things like this: 

myArray[0] = 1
console.log( myArray[7] )
// or even:
myArray.length = 42

But the really powerful thing about arrays is that we can also use variables as our indexes. So for example: 

var array = [ "breakfast", "lunch", "tea", "dinner" ]
var i = 0
console.log( array[i] ) // what will this print?
i = i + 1
console.log( array[i] ) // what will this print now?

// think about when this assignment happens:
i = array.length - 1
 // so this does what you'd expect:
console.log( array[i] )
// but if now we add another value:
array.push("dessert")
// what is the value of i now?
console.log( array[i] )

This gets exciting when we change the index variable in interesting ways. 

// We've already seen how to print an array like this:
console.log( array )

// But here's another way:
var i = 0
while ( i < array.length ) {
  console.log( "Meal #" + i + " is: " + array[i] )
  i = i + 1
}
